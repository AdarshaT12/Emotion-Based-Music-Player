# Emotion Based Music Player

Emotion Based Music Player is a Python application that detects the user's facial emotion using webcam and automatically plays music based on detected emotion.

## Features

- Real-time facial emotion detection
- Automatic music selection
- Webcam-based emotion recognition
- Simple and interactive interface

## Technologies Used

- Python
- OpenCV
- FER (Facial Emotion Recognition)
- Pygame

## Project Structure

Emotion-Based-Music-Player
│
├── main.py
├── requirements.txt
├── songs/
└── assets/

## Installation

Clone the repository

git clone https://github.com/adarshaT12/Emotion-Based-Music-Player.git

Install dependencies

pip install -r requirements.txt

Run the program

python main.py

## How It Works

1. Webcam captures the user's face.
2. Emotion detection model analyzes facial expression.
3. Detected emotion triggers music playback.
